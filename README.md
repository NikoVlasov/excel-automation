# Excel Automation Project

Что делает проект?
Автоматически собирает данные из нескольких Excel-файлов и считает итоги.

# Как запустить?
1. Скачайте `DataProcessor.xlsm`.
2. Положите файлы в `input_data`.
3. Откройте Excel → нажмите `Alt+F8` → выберите `GenerateReport`.

# Результат
 Объединение данных из множества Excel-файлов в единую таблицу
 Автоматический расчет итоговых показателей
 Готовая структура для дальнейшего анализа (сводные таблицы, графики)
